利用正则表达式判断输入的数据是否符合格式。getPoly和getArray对输入的数据进行处理。主函数为先输入数据，删除所有的空格，判断加减，再调用add和sub函数进行计算，最后输出结果。可进行多次计算。

我是用idea编写的，但是idea中的\b不能正常使用。建议使用eclipse进行检查。

数组长度为10000；
计算结果为0的最终会输出{()}；
{(1,2),()}不会报错，但是无法进行正常运算，输出只会显示{(1,2)},即包含（）的{}中其他所有项；
{(1,2),(,),(12,3)}-{(-1,2)}会报错，（,）；
{(1,	5)} 包含tab的会报错；
